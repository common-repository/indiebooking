Indiebooking gives you the ability to automate your booking process.
== Description ==

Indiebooking offers a simple and fast possibility for self-marketing of real properties / real estate.
It is irrelevant whether the property is offered in the long or short stay area.

The installation is done easily on your Wordpress-Installation. In order to use the indiebooking-Plugin you also need to install one of our indiebooking-themes. You can find them on www.indiebooking.de

= The indiebooking-Plugin offers you everything you need for marketing right from the start, as well as much more: = 
* Maintain and manage 3 objects at multiple locations
* Fully automated booking process with invoice creation and E-Mailing.
* VAT Split in accounting
* Selectable arrival days
* Determination of the minimum booking period
* Overview of all bookings
* Specify the maximum time that is available for the booking process.
* Storing cancellation conditions that are automatically applied.
* Specify individual e-mail texts

== Installation ==

How to install the plugin and get it working?

= Using The WordPress Dashboard =
1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'Indiebooking'
3. Click 'Install Now'
4. Activate Indiebooking on the Plugin dashboard
5. Specify youre Settings under the 'Indiebooking->Settings' page.

= Using FTP =
1. Download 'indiebooking.zip'
2. Extract the 'indiebooking' directory to your computer
3. Upload the 'indiebooking' directory to the `/wp-content/plugins/` directory
4. Activate Indiebooking in the Plugin dashboard

= Theme support =
To use the indiebooking-Plugin you also need to install one of our indiebooking-Themes to use all functionalities like showing what you have for rent, the avalibility-calendar, the Booking- and payment-process and so on. You can find indiebooking-themes on www.indiebooking.de for free download.
